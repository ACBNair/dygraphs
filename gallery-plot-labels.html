Plot Labels

You can add labels to a dygraph using the main, ylab and xlab arguments. For example:

dygraph(discoveries, 
        main = "Important Discoveries", 
        ylab = "Discoveries / Year")

Itâ€™s also possible to specify labels as part of a dyAxis definition. For example:

dygraph(discoveries, main = "Important Discoveries") %>%
  dyAxis("y", label = "Discoveries / Year")

The appearance of labels can be customized extensively using CSS. See the CSS Label Styling example for additional details.
